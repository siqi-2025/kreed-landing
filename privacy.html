扩展权限说明

今日知识地图请求以下浏览器权限，每项权限均有明确的功能必要性：

权限用途必要性说明
tabs 获取当前标签页的 URL 和标题 核心功能：记录阅读页面、生成知识地图的必要输入
activeTab 访问当前活跃标签页的内容 AI 摘要：读取页面正文生成摘要和关键词
storage 本地数据存储 存储阅读记录、用户设置和积分数据
alarms 定时任务调度 自动清理过期数据、定时统计更新

我们不请求 history、bookmarks、cookies 等敏感权限。所有数据均存储在您的浏览器本地。

一适用范围

本隐私政策适用于浏览器扩展「今日知识地图」（KnowledgeMap）。我们承诺仅将您授权的数据用于提升您的个人阅读体验，不会将您的个人数据用于广告定向或出售给第三方。

我们的承诺：我们收集的每一项数据都有明确的产品用途，不会将您的阅读数据用于任何广告、分析或商业变现目的。

Limited Use Compliance: The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

有限使用合规声明：从 Google API 获取的信息的使用将遵守 Chrome Web Store 用户数据政策，包括有限使用要求。

二我们收集什么

当您使用今日知识地图时，我们会在您知情并主动使用的范围内收集以下数据：

数据类型收集方式用途
访问记录
（URL、标题、时间）		 您访问网页时自动记录 生成阅读统计和知识地图
阅读时长 标签页活跃时计时 计算阅读价值积分（RVC）
文章正文
（用于 AI 摘要）		 读取页面 DOM 内容 生成 AI 摘要和关键词提取
主题分类标签 基于正文关键词自动分类 构建兴趣图谱和推荐
用户设置
（AI API Key 等）		 您主动在扩展设置中填写 配置 AI 摘要功能

我们不收集：密码、信用卡信息、Cookie、浏览历史（仅记录当前会话的阅读页面）、表单输入内容。

三数据如何存储

本地存储（主要方式）

您的所有阅读数据默认存储在您自己的设备上，使用浏览器的本地存储空间（IndexedDB）。我们没有也不会架设任何云端服务器来存储您的个人阅读数据。

数据加密

如果您配置了 AI 摘要功能，扩展与 AI 服务之间的通信通过 HTTPS 加密传输，防止中间人攻击。

数据隔离

您的数据存储在浏览器为扩展分配的独立存储空间中，与其他扩展和网站的数据完全隔离。

注意：卸载扩展时，本地存储的数据将自动清除。在卸载前，建议您使用导出功能备份重要数据。

服务数据内容用途隐私政策

智谱 GLM API
（AI 摘要/分类）		 文章正文片段

我们不会将您的数据发送给任何广告平台、数据分析服务商或社交媒体平台。

如您使用最大化版（Max 版）并主动配置 n8n 工作流同步功能，您需要自行确保 n8n 服务的数据安全，此时数据传输由您自行控制的服务器处理。

五您的权利

您对您的个人数据拥有完整的控制权：

  1. 知情权：本隐私政策完整披露了我们的数据收集和使用方式。
  2. 访问权：您可以随时通过扩展的 Dashboard 查看所有已记录的数据。
  3. 导出权：您可以使用导出功能将所有数据导出为 JSON 或 CSV 文件。
  4. 删除权：您可以随时删除单条记录、清空特定日期的数据，或一键清空所有数据。
  5. 拒绝权：您可以随时停用扩展，所有追踪立即停止。
  6. 投诉权：如您认为我们侵犯了您的隐私权，您有权向当地数据保护机构投诉。

六数据保留

我们采用智能分层存储策略，自动管理数据保留：

数据类型保留时间说明
高价值内容
（深度阅读/收藏）		 永久 完整存储，含 AI 摘要
中等价值内容 1 年 含 AI 摘要和关键词
低价值内容
（浅读/新闻）		 3 个月 基础元数据，压缩存储
极低价值内容 1 天 仅 URL 哈希（去重用）

您可以在设置中切换存储模式（极简/平衡/完整），调整自动清理规则。
